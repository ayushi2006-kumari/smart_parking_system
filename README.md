# smart_parking_system
A simple and efficient Smart Parking System built using HTML, CSS, and JavaScript to manage vehicle parking with features like car entry, exit, admin control, and special slots for VIP and emergency vehicles.

📌 Features
🚘 User Module
Car Entry System (park vehicle)
Car Exit System (free slot)
Real-time Slot Availability
Automatic Slot Allocation
👑 Admin Panel
View all parked vehicles
Monitor slot status (Available / Occupied)
Add / Remove parking slots
Manage VIP & Emergency slots
today ,this month and total revenue
🚨 Special Slots
VIP Parking (reserved for special users)
Emergency Parking (ambulance, police, etc.)
Priority-based slot allocation
🛠️ Tech Stack
HTML – Structure
CSS – Styling
JavaScript – Functionality & Logic
